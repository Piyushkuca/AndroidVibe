**Vive** is an open-source Android application that seamlessly blends an intelligent AI chat interface with a dedicated "vibe coding" environment. Dive into conversational coding, brainstorm logic, and let the AI assist you in building out ideas effortlessly directly from your device.

---

## ✨ Features

* **💬 Intelligent AI Chat:** Engage with an advanced AI for brainstorming, troubleshooting, and general inquiries in a fluid chat interface.
* **💻 Vibe Coding Environment:** A tailored experience for generating, reviewing, and refining code snippets on the go. Focus on the creative flow while the AI handles the syntax.
* **📱 Native-Feel Android App:** Built to run smoothly on Android devices, including tablets, providing a responsive and intuitive mobile coding experience.
* **🔓 100% Open Source:** Fully open to the community. Inspect the code, suggest improvements, and build upon the foundation.

## 🛠️ Tech Stack

* **Framework:** Kotlin
* **Platform:** Android
* **AI Integration:** REST/WebSocket APIs for LLM communication

## 🚀 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

* [Flutter SDK](https://docs.flutter.dev/get-started/install) installed on your system.
* An Android emulator or a physical Android device connected via USB/Wi-Fi debugging.
* Your preferred IDE (e.g., VS Code or Android Studio).

### Installation

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/Piyushkuca/AndroidVibe.git](https://github.com/Piyushkuca/AndroidVibe.git)
    cd vive
    ```

2.  **Fetch dependencies:**
    ```bash
    flutter pub get
    ```

3.  **Set up API Keys (if applicable):**
    * Rename `.env.example` to `.env`.
    * Add your preferred LLM API keys to the `.env` file.

4.  **Run the application:**
    ```bash
    flutter run
    ```
